# ruby-tictactoe

The Odin Project - Tic Tac Toe from Ruby curriculum

### Play TicTacToe on your Command Line!

To play, simply install the gem 'rutictactoe'

```ruby
gem install rutictactoe
```

And then, just require the gem on your ruby file!

```ruby
# app.rb
require "rutictactoe"
```
